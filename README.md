# Custom Pearl

An AI-powered customized e-commerce platform built using modern web technologies.

## Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- SQL Server (Phase 1)
- Firebase Authentication

## Features

- User Authentication
- Product Catalog
- Product Customization
- Shopping Cart
- Checkout
- Admin Dashboard
- WhatsApp Integration

## Project Status

🚧 Phase 1 - SQL Server Version

Phase 2 will migrate the database to Firebase Firestore and deploy the application.
